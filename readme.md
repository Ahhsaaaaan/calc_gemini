# Gemini Calculator

A simple and beautiful calculator application built with HTML, CSS, and JavaScript.

## Features

*   Basic arithmetic operations (+, -, *, /, %)
*   Dark and light theme toggle
*   Responsive design
*   Glowing buttons on hover
*   History of the current operation

## How to Use

1.  Clone the repository:
    ```bash
    git clone https://github.com/Ahhsaaaaan/calc_gemini.git
    ```
2.  Open the `index.html` file in your web browser.

## Technologies Used

*   HTML
*   CSS
*   JavaScript
